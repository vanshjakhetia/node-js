NOTES APP

This Notes app(note taking app) has various features like adding an note, deleting an note, listing out all the note, reading an note.
This app is build using node.js and some NPM modules like nodemon(version - 2.0.22), Yargs(version - 12.0.2) , Validator(version - 10.8.0), Chalk(version - 2.4.1).
